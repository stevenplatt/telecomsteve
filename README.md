# telecomsteve

Source code for my personal website, [telecomsteve.com](https://telecomsteve.com).

Built with React, TypeScript, Vite, and Chakra UI. The app lives in the
[`telecomsteve/`](telecomsteve/) directory.

## Getting started

```bash
cd telecomsteve
npm install
npm run dev
```

Then open http://localhost:5173.

## Scripts

Run these from the `telecomsteve/` directory:

| Command | Description |
| --- | --- |
| `npm run dev` | Start the dev server |
| `npm run build` | Type-check and build for production |
| `npm run lint` | Lint the code |
| `npm test` | Run unit tests (Vitest) |
| `npm run test:e2e` | Run end-to-end tests (Playwright) |
| `npm run verify` | Lint, type-check, unit test, and build |
